描述：
输入一个整数，将这个整数以字符串的形式逆序输出
程序不考虑负数的情况，若数字含有0，则逆序形式也含有0，如输入为100，则输出为001

题目来源:	 内部整理 
练习阶段:	 初级 
运行时间限制:	10Sec
内存限制:	128MByte
输入:	 
输入一个int整数
 
输出:	 
将这个整数以字符串的形式逆序输出
 
样例输入:	
1516000

                   
样例输出:	
0006151
