描述: 	
正整数A和正整数B 的最小公倍数是指 能被A和B整除的最小的正整数值，设计一个算法，求输入A和B的最小公倍数。
 
知识点:	 循环,位运算 
题目来源:	 内部整理 
练习阶段:	 初级 
运行时间限制:	10Sec
内存限制:	128MByte
输入:	 
输入两个正整数A和B。
 
输出:	 
输出A和B的最小公倍数。
 
样例输入:	
5 
7

                   
样例输出:	
35

                    
答案提示:	[辗转相除法求最大公约数，最小公倍数=两整数的乘积÷最大公约数]
