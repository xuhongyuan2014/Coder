最近邻居
Time Limit: 10000/5000 MS (Java/Others) Memory Limit: 65536/65536 K (Java/Others)
Problem Description:
有n个点随机分布在原点周围，找出其中距离最近的两个点。输入是n个点的<x,y>坐标数组（points[]），算出距离最近的两个点，把这两个点在输入数组中的下标从小到大输出。

注意：
1. 我们的输入数据是按点与原点之间的距离从小到大排序的。输入的数据点可能会很多，所以要求速度越快越好。提示：平面上任意a，b，c三个点，D(a,b)>=D(a,c)-D(b,c)。
2. 请尽量使用double双精度类型计算，避免浮点精度误差
3. 测试数据保证解的存在性和唯一性

（提示：请您务必点击此处，下载参考框架）
输入
数字N，#表示有多少个点 N < 2000000
浮点数X 浮点数Y #第0个点的坐标
...
浮点数X 浮点数Y #第N-1个点的坐标
输出
Closest points: p0, p1 # p0, p1是距离最近两个点的的编号，p0 < p1。

样例输入
3
1.0 1.0002
3.03 3.023
0.0 -0.001
样例输出
Closest points: 0, 2
